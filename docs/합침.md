1. 집합(Set) 및 인덱스 정의$\mathcal{T}$ : 창고 내 보관 랙에 꽂혀 있는 전체 토트(Tote)의 집합 (인덱스 $j \in \mathcal{T}$)$K$ : 시스템 내 조립 대기 및 처리 중인 활성 키트(Kit)의 집합 (인덱스 $i \in K$)$M$ : 다품종 부품 모델(Model/SKU)의 집합 (인덱스 $m \in M$)$S$ : 작업이 수행되는 실시간 키팅 스테이션(Kitting Station)의 집합 (인덱스 $s \in S$)$E$ : 시뮬레이션 전체 운영 기간 동안 발생한 디스패칭 의사결정 이벤트의 집합 (인덱스 $e \in E$)$A$ : 제어 대상인 무인운반차(AGV)의 집합 (인덱스 $a \in A$)2. 전역 목적함수 (Global Objective Function)$$Minimize \quad Z = 0.6 \cdot \hat{T} + 0.3 \cdot \hat{F} + 0.1 \cdot \hat{D}$$최종 납기 지수 $\hat{T}$를 메인 평가지표($0.6$)로 잡고, 창고 운영 지속 효율성을 위한 파편화 제어 지수 $\hat{F}$($0.3$)와 AGV의 배터리 및 이송 자원 효율 지수 $\hat{D}$($0.1$)를 선형 결합함.① $\hat{T}$ : 총 납기 지연 지표 (Normalized Total Tardiness)$$\hat{T} = \begin{cases} \frac{\sum_{i \in K} \left( \max(0, C_i - d_i) \right)^2}{T_{\text{greedy}}}, & \text{if }T_{\text{greedy}} > 0 \\ 0, & \text{if }T_{\text{greedy}} = 0 \end{cases}$$$C_i$ : 키트 $i$의 실제 완성 조립 시각 (s)$d_i$ : 키트 $i$의 납기 마감 시각 (s)$T_{\text{greedy}}$ : "가장 급한 납기 마감 + 최대 부품 수량 즉시 공급" 규칙(Greedy Policy) 하에 시뮬레이션을 수행했을 때 발생하는 총 지연 시간특이사항 : 강화학습 에이전트의 학습 초기 단계 및 부적합한 GA 염색체 평가 시에는 결과가 Greedy Baseline보다 나쁠 수 있어, $\hat{T}$ 지표값이 $1.0$을 초과하여 폭발할 수 있음② $\hat{F}$ : 창고 파편화 지표 (Fragmentation Index)$$\hat{F} = \frac{\sum_{j \in \mathcal{T}} f_{\text{score}}(j)}{\vert{}\mathcal{T}\vert{}}$$개별 토트 파편화 점수 함수 $f_{\text{score}}(j)$:$$f_{\text{score}}(j) = \begin{cases} 0, & \text{if } \sum_{m \in M} q_{jm} = 0 \\ w_{f1} \left( 1 - \frac{\sum_{m \in M} V_{\text{box\_total}, jm}}{V_{\text{max}}} \right) + w_{f2} \left( \frac{\sum_{m \in M} V_{\text{dead}, jm}}{V_{\text{max}}} \right), & \text{if } \sum_{m \in M} q_{jm} > 0 \end{cases}$$$V_{\text{box\_total}, jm} = \left\lceil \frac{q_{jm}}{L_m} \right\rceil \cdot v_{\text{box}, m}$ (토트 내 특정 부품 m의 소박스들이 실제 차지하는 물리적 체적 합산)$V_{\text{dead}, jm} = \left( \left\lceil \frac{q_{jm}}{L_m} \right\rceil \cdot L_m - q_{jm} \right) \cdot v_m$ (소박스 내 빈 공간으로 발생하는 데드 스페이스 체적 합산)$\text{subject to } \quad w_{f1} + w_{f2} = 1.0$ (체적 점유 항과 데드 스페이스 발생 항의 민감도 가중비)$q_{jm}$ : 토트 $j$가 보관 중인 부품 $m$의 실제 재고량$L_m$ : 부품 $m$의 표준 소박스 입고 단위 (Lot Size 상수)$v_m$ : 부품 $m$의 개당 순수 체적 ($\text{cm}^3$)$v_{\text{box}, m}$ : 부품 $m$의 표준 소박스 1개의 외부 체적 ($\text{cm}^3$)$V_{\text{max}}$ : 토트 내부의 물리적 최대 적재 가능 수용 체적 한계점 ($\text{cm}^3$)③ $\hat{D}$ : AGV 이동 거리 지표 (Distance Index)$$\hat{D} = \frac{\sum_{e \in E} D_{\text{actual}}(e)}{\vert{}E\vert{} \cdot D_{\text{max}}}$$$D_{\text{actual}}(e)$ : $e$번째 디스패칭 결정 결과에 따라 AGV가 수행하는 맨해튼 주행 거리 ($\text{공차 이동 거리} + \text{실차 왕복 이동 거리}$)$$D_{\text{actual}}(e) = (\vert{}x_{\text{AGV}, e} - x_{\text{Storage}, e}\vert{} + \vert{}y_{\text{AGV}, e} - y_{\text{Storage}, e}\vert{}) + 2 \times (\vert{}x_{\text{Storage}, e} - x_{\text{Station}, e}\vert{} + \vert{}y_{\text{Storage}, e} - y_{\text{Station}, e}\vert{})$$$D_{\text{max}}$ : 의사결정당 랙-스테이션 기하구조 상 물리적으로 발생 가능한 최악의 맨해튼 주행 한계값$$D_{\text{max}} = 3 \times (X_{\text{max}} + Y_{\text{max}})$$$\vert{}E\vert{}$ : 시뮬레이션 종료 시까지 생성 및 처리 완료된 디스패칭 의사결정 이벤트 총수3. 실시간 디스패칭 우선순위 결정: CPR Score시뮬레이션 구동 중 임의의 가용한 AGV $a \in A$, 매칭 가능한 토트 $j \in \mathcal{T}$, 실시간 요구 키트 $i \in K$ 조합이 들어올 때마다 아래의 점수를 산출하여 가장 스코어가 높은 조합을 매칭함.$$\text{CPR}(a, j, i) = \alpha_1 \cdot S_T(j, i) - \alpha_2 \cdot \Delta S_F(j, i) - \alpha_3 \cdot S_D(a, j, i)$$$$\text{subject to} \quad \alpha_1 + \alpha_2 + \alpha_3 = 1.0$$① 납기 점수 ($S_T$)$$S_T(j, i) = \text{Urgency}(i) + \gamma \cdot \text{Progress}(j, i)$$키트 $i$의 시간적 촉박도($\text{Urgency}$)를 근본 체급으로 두며, 해당 토트 $j$가 키트 $i$의 오더를 얼마나 한번에 채워주는지($\text{Progress}$)를 보너스 버프로 결합하는 합연산식.1) 긴급도 항 ($\text{Urgency}$):$$\text{Urgency}(i) = \begin{cases} \epsilon + \beta \cdot \left( \frac{t_{\text{now}}}{d_i - \Delta t_{\text{margin}}} \right), & \text{if } t_{\text{now}} < d_i - \Delta t_{\text{margin}} \quad \text{(평상시)} \\ (\epsilon + \beta) + (1.0 - \beta) \cdot \left( 1 - e^{-\lambda \cdot x_i} \right)^p, & \text{if } t_{\text{now}} \ge d_i - \Delta t_{\text{margin}} \quad \text{(데드라인 임박 시)} \end{cases}$$$t_{\text{now}}$ : 현재 시뮬레이션 타임스탬프 (s)$d_i$ : 키트 $i$의 납기 마감 시각 (s)$\Delta t_{\text{margin}}$ : 긴급도를 가속하여 급격히 증가시키기 위한 완충 한계시간 버퍼 (s)$x_i = t_{\text{now}} - (d_i - \Delta t_{\text{margin}})$ : 안전마진 진입 이후 누적된 경과 시간 (s)$\epsilon$ : 평상시 극초반 단계에서 점수가 완전히 $0$에 수렴하는 현상을 막기 위한 최소 하한값$\beta$ : 평상시 단계에서의 납기 선형 상승 폭을 통제하는 제한 상한치$\lambda$ : 2단계 지수 포화 곡선의 기울기 수렴 속도 제어 민감도 상수$p$ : 마감 임박 시점에 지수 수식을 한 차례 더 가속화하여 수직 상승 댐퍼를 형성시키는 가속 지수2) 진행률 항 ($\text{Progress}$):$$\text{Progress}(j, i) = \begin{cases} \frac{\sum_{m \in M} \min(r_{im}, q_{jm})}{\sum_{m \in M} r_{im}}, & \text{if } \sum_{m \in M} r_{im} > 0 \\ 0, & \text{if } \sum_{m \in M} r_{im} = 0 \end{cases}$$$r_{im}$ : 현재 의사결정 시점 기준 키트 $i$가 요구하는 부품 $m$의 잔여 오더 요구량$q_{jm}$ : 해당 토트 $j$가 현재 실시간으로 보관하고 있는 부품 $m$의 보유 재고량3) 미세 가중치 ($\gamma$):두 키트의 납기 마감 시각 차이가 일정 수준 벌어져 있을 때, 진행률 편차에 의해 더 급한 오더가 새치기당하지 않도록 차단하는 가중 밸런서.허용하고자 하는 역전 마지노선 임계 격차값($\Delta U_{\text{threshold}}$)을 넘을 수 없도록 $\gamma < \Delta U_{\text{threshold}}$ 조건을 만족하도록 설계함.② 파편화 점수 ($\Delta S_F$)$$\Delta S_F(j, i) = f_{\text{score}}(j') - f_{\text{score}}(j)$$자재 피킹 작업 실행 전과 예상 실행 후의 개별 토트 단위 파편화 점수 변화량.$f_{\text{score}}$ : 전역 목적함수 $\hat{F}$ 내부에 정의된 동일한 소박스/데드스페이스 낭비 측정 점수 수식$j$ : 현재 피킹 작업 전의 토트 $j$의 재고 상태$j'$ : 토트 $j$가 키트 $i$의 요구 부품을 충족시키기 위해 자재를 매칭 차감한 직후의 예상 잔여 상태토트 $j'$의 잔여 재고는 $q'_{jm} = q_{jm} - \min(r_{im}, q_{jm})$ 로 수식 정합성을 만족함.③ 이동거리 점수 ($S_D$)$$S_D(a, j, i) = \frac{D_{\text{actual}}(a, j, i)}{D_{\text{max}}}$$실시간 매칭 결정 시 발생하는 해당 AGV $a$의 이송 비용을 목적함수 최악 주행 거리로 나누어 정규화($[0, 1]$)한 물리 페널티 점수.$D_{\text{actual}}(a, j, i)$ : AGV $a$가 토트 $j$의 보관 랙 주소를 찍고 키트 $i$가 배정된 조립 스테이션까지 최종 이동 및 복귀하는 데 따르는 실제 맨해튼 이송 거리:$$D_{\text{actual}}(a, j, i) = \left( \vert{}x_a - x_j\vert{} + \vert{}y_a - y_j\vert{} \right) + 2 \times \left( \vert{}x_j - x_{s(i)}\vert{} + \vert{}y_j - y_{s(i)}\vert{} \right)$$$s(i) \in S$ : 키트 $i$가 할당되어 현재 오더를 수급하고 있는 대상 키팅 스테이션 $s$의 위치 좌표 $(x_{s(i)}, y_{s(i)})$
